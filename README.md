# WiSED datasets and figure source data

This repository contains the data needed to reproduce the retained WiSED
engineering validations and the numerical source data supporting their
manuscript figures. The associated source code is maintained in
[`Hust-WiSED`](https://github.com/HUSTOROP/Hust-WiSED).

## Contents

| Directory | Contents | Provenance |
| --- | --- | --- |
| `sib_diffusion/` | Generated radial concentration field for the retained 3% noise SIB experiment. | Generated by the SIB simulation in `Hust-WiSED`. |
| `soc_eis/` | Public raw EIS and capacity measurements for batteries B01-B11. | Mustafa *et al.*, Mendeley Data V2. |
| `figure_data/` | CSV/JSON/Markdown numerical source data for the retained SIB and B11 SOC-EIS figures. | Derived from the documented WiSED analyses. |

## Use with the code repository

Clone both repositories, then run this command from the `Hust-WiSED` root:

```bash
python scripts/install_release_data.py --dataset-root ../Hust-WiSED-Dataset
```

The script installs only the field and raw measurement files needed by the two
engineering runs. It does not alter this dataset repository.

## SOC-EIS source dataset

The raw `soc_eis/` directory preserves the original B01-B11 battery and test
cycle layout. It was obtained from:

Mustafa, H., Bourelly, C., Vitelli, M., Milano, F., Molinara, M. and Ferrigno,
L. *SoC Estimation on Li-ion Batteries: A New EIS-based Dataset for data-driven
applications*. Mendeley Data, V2 (2024). DOI:
[10.17632/cb887gkmxw.2](https://doi.org/10.17632/cb887gkmxw.2).

The associated data article is Mustafa *et al.*, *Data in Brief* **57**, 110947
(2024). DOI: [10.1016/j.dib.2024.110947](https://doi.org/10.1016/j.dib.2024.110947).
Please cite both records and retain the original CC BY 4.0 attribution when
reusing any SOC-EIS measurement or derivative result.
