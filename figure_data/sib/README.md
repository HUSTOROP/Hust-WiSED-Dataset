# SIB figure source data

These files contain equation-assessment and downstream-application metrics for
the retained SIB manuscript example.

- `sib_diffusion_application_metrics.csv`: numerical values used for downstream
  field-reconstruction and future-rollout assessment.
- `sib_diffusion_equation_assessment.json`: machine-readable equation and
  metric summary.
- `sib_diffusion_equation_assessment.md`: human-readable summary of the same
  assessment.

The corresponding generated concentration field is
`../../sib_diffusion/sib_diffusion_raw_field_noise0.03.npz`. The generator,
post-processing code and figure scripts are in the companion `Hust-WiSED`
repository. Cite WiSED when reusing these project-derived data.
