# SOC-EIS B11 figure source data

This directory retains the manuscript's representative held-out B11 result.

- `soc_eis_b11_soc_predictions.csv`: 38 SoC estimates obtained by matching
  measured B11 spectra to the WiSED-generated impedance trajectory.
- `soc_eis_b11_soc_metrics.csv`: numerical B11 SoC-evaluation metrics.
- `soc_eis_b11_summary.json`: discovered coupled equations and B11 test
  metrics.

The underlying public measurements are in `../../soc_eis/`. Cite Mustafa
*et al.* Mendeley Data V2 (DOI 10.17632/cb887gkmxw.2) and the associated
*Data in Brief* article (DOI 10.1016/j.dib.2024.110947), in addition to the
WiSED manuscript/software release.
