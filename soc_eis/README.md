# Public SOC-EIS dataset

This directory contains the public EIS and capacity measurements used by the
SOC-EIS example. The files retain the authors' B01-B11 battery and test-cycle
directory layout; this README is the only added file.

## Source and citation

Mustafa, H., Bourelly, C., Vitelli, M., Milano, F., Molinara, M. and Ferrigno,
L. *SoC Estimation on Li-ion Batteries: A New EIS-based Dataset for data-driven
applications*. Mendeley Data, V2 (2024). https://doi.org/10.17632/cb887gkmxw.2

The associated data article is:

Mustafa, H., Bourelly, C., Vitelli, M., Milano, F., Molinara, M. and Ferrigno,
L. SoC estimation on Li-ion batteries: A new EIS-based dataset for data-driven
applications. *Data in Brief* **57**, 110947 (2024).
https://doi.org/10.1016/j.dib.2024.110947

The dataset is publicly available at
https://data.mendeley.com/datasets/cb887gkmxw/2 and is released under the
CC BY 4.0 license. Please cite both records when reusing these measurements.
